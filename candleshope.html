E-Commerce Solution

Premium Boutique Candle Store

CandleShop is a premium, full-stack e-commerce application tailored for a boutique candle business. It is designed to provide a seamless shopping experience for customers and a robust management system for administrators.

Next.js 14 Speed

Project Gallery

Core Features

Customer Experience

  • Elegant Storefront: Browse hand-poured candles by category or scent profile.
  • Advanced Search: Real-time filtering by scent, price, and availability.
  • Smooth Checkout: Multi-step process with saved addresses and coupons.
  • Order Tracking: Real-time status updates and purchase history.

Administrative Tools

  • Inventory Management: Stock tracking with automated low-stock alerts.
  • Order Fulfillment: Dedicated dashboard to manage delivery statuses.
  • Product Control: Full CRUD operations via intuitive admin UI.

Secure Infrastructure

  • Auth: Secure social login via Google OAuth 2.0.
  • Payments: Integrated Razorpay for production-grade transactions.
  • Data Security: Row Level Security (RLS) via Supabase policies.
